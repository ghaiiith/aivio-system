# AIVIO Residence Community

Adds the resident profile layer above AIVIO Real Estate units for mobile apps.

Main models:

- `aivio.resident.profile`
- `aivio.family.member`
- `aivio.resident.vehicle`
- `aivio.ads.banner`

Resident portal users are linked to their allowed units through `res.users.aivio_unit_ids`.
