# AIVIO Mobile API Gateway

REST API module for the AIVIO Resident, Security, and Maintenance mobile apps.

Docs are served after installing the module:

- `/api/aivio/swagger.json`
- `/api/aivio/openapi.json`
- `/api/aivio/openapi.yaml`
- `/api/aivio/postman.json`

Authentication uses `Authorization: Bearer <token>` returned by `/api/auth/login`.
